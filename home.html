Welcome!

This is just a simple webtool to help locate Nikke game files.
If you're unfamiliar with how to use this tool, please check out How to Use in the navigation bar.

Addressables are now encrypted. Please refer to the Folder section of "How to Use" to obtain the decrypted files

Finders

Common Files
- This tool can be used to locate commonly used games files, such as : Aim / Cover / Standing / Portraits / Burst / SD-Model / Background / etc.
- Most of these files are located in the 'Mod' folders at: %USERPROFILE%\AppData\LocalLow\Unity\com_proximabeta_NIKKE\.
- HD = High quality / SD = Low quality

Physics Files
- This tool is specifically designed to locate the 'Physics Bundle File', which contains all character physics data.
- 'Containers' refer to directories within the Physics Bundle File and are not actual files found in the mod folder.

Voice Files
- This tool is designed to locate voice-related files, including : Character Voices / Voices (Max Bond) / Title screen Voices.

Options

- If you want to easily clear all saved data or if you majorly screwed up and cannot interact with the finders anymore, use this.
