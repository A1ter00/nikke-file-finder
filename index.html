Nikke File Finder by A1teri - Home Finders - Common Files (HD) - Common Files (SD) - Physics Files Extra - Character ID's - How To Use

Instructions

  1. Select which finder to use in the Navigation Bar.
    • Common / Physics
  2. Upload the folder that contains your Addressable files
    • (Refer to the Folders section)
  3. Select which the type of file you're looking for (All, Aim,Standing, etc.)
    • Export List can be use to save data into csv files
  4. Search using character Name or ID (Optional)
    • Ex :
      - '010 / Rapi' will show the same result
      - '10' will show all characters that has '10' in their IDs (Rapi/Laplace/Drake/etc)
      - 'ra' will show all characters that has 'ra' as part of their name (Rapi/Rapunzle/Sakura/etc)
