Nikke File Finder by A1teri Finders - Upload Files - Common Files - Physics Files - Voice Files - Manual Search Extra - Character ID's - How To Use

Instructions

  1. Select which finder to use in the Navigation Bar.
    • Common / Physics / Voice Files - Are used to locate commonly used files
    • Manual Search - Use this if what youre looking for arent listed in the other finders
  2. Upload the folder that contains your Addressable / Voice files
    • (Refer to the Folders section)
  3. Select which the type of file you're looking for (All, Aim,Standing,Voice, etc.)
    • Export List can be use to save data into csv files
  4. Search using character Name or ID (Optional)

Folders

  1. Addressable Directory
    • %USERPROFILE%\AppData\LocalLow\com_proximabeta\NIKKE\com.unity.addressables
  2. Mod Directory
    • %USERPROFILE%\AppData\LocalLow\Unity\com_proximabeta_NIKKE\HERE
  3. Voice Directory
    • %USERPROFILE%\Appdata\LocalLow\com_proximabeta\NIKKE\saus

Reminders

Before uploading your addressable folder.

  1. Make sure you disable all mods applied by NMM
  2. Delete the addressable folder then run the game once to obtain fresh files

Tips

Every update, you can follow the steps mentioned in the Reminders section then make a backup of your addressables to skip repeating the said process

About

This is just a simple webtool to help locate Nikke game files.
