File Finder Physics Finder HOW TO USE

Instructions

  1. Upload a file or folder
    • Open a specific file (.json files)
    • Upload the entire addressable folder (Recommended)
  2. Select which the type of file you're looking for (None, All, Aim,Standing,Voice, etc.)
    • 'All' displays every file
    • Export Table can be use to save data into csv files; (Disabled on All)
  3. Search using character ID's (Optional)
