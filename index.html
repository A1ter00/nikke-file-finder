File Finder Physics Finder Miscellaneous Finder HOW TO USE

Instructions

  1. Upload the entire addressable folder
    • (Refer to the Folders section)
  2. Select which the type of file you're looking for (All, Aim,Standing,Voice, etc.)
    • All displays every file
    • Export Table can be use to save data into csv files
  3. Search using character Name or ID (Optional)

Folders

  1. Addressable Directory
    • C:\Users\USERNAME\AppData\LocalLow\ com_proximabeta\NIKKE\com.unity.addressables
  2. Mod Directory
    • C:\Users\USERNAME\AppData\LocalLow\ Unity\com_proximabeta_NIKKE\ HERE

Reminders

Before uploading your addressable folder.

  1. Make sure you disable all mods applied by NMM
  2. Delete the addressable folder then run the game once
This will ensure you only have the latest version of the files preventing duplicates and errors while using the tool.

Tips

Every update, you can follow the steps mentioned in the Reminders section then make a backup of your addressables to skip repeating that process whenever you use the tool

About

This is a collection of simple tools used to locate Nikke files.

  1. File Finder
    • Used to locate commonly used files.
  2. Physics Finder
    • Used to locate character physics related files.
  3. Miscellaneous Finder
    • Can be used to locate most of the game files.
